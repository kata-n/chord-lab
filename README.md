# 🎹 コード進行ラボ

音楽のコード進行を「読んで・聴いて・解いて」学べる、初心者向けのWebアプリです。

**公開URL: https://kata-n.github.io/chord-lab/** (スマホからもどうぞ)

## 起動方法

```bash
npm install   # 初回のみ
npm run dev   # http://localhost:5173 で起動
```

公開用にビルドする場合は `npm run build`(`dist/` に出力)。

## デプロイ

`main` ブランチに push すると、GitHub Actions(`.github/workflows/deploy.yml`)が自動でビルドして GitHub Pages に公開します。手動で反映したい場合も `git push` するだけです。

## 機能

- **📖 学ぶ** — 11のレッスン(三和音の基礎 → ダイアトニックコード → コードの機能 T/S/D → 定番進行 → セブンスコード → マイナーキー → sus4/add9/dim7 などの多彩なコード → 丸サ進行とブラックアダーコード → アイドル曲の定番パターン → 泣きのコード(借用和音) → 転調体験)。例はすべてピアノ音で再生でき、鍵盤図に構成音がハイライトされます。ヘッダーのキー選択で8つのメジャーキー(+その平行短調)に移調できます。
- **👂 聴く** — イヤートレーニング。メジャー/マイナーの聴き分け、コードの種類あて(sus4・aug・dim7など)、定番進行・アイドル進行の聴き当て、そして進行を1コードずつ度数で聴き取る**ディクテーション**(3段階のレベル、ベースだけ再生モード付き)。
- **✏️ クイズ** — 構成音・ディグリー・マイナーディグリー・進行の穴埋め・機能のドリル。
- **📊 記録** — レッスン読了とクイズ成績を localStorage に保存。連続正解記録つき。

## 技術構成

- React 18 + Vite
- 音は Web Audio API のオシレーターで合成(外部音源・ライブラリ不使用)
- 主要なロジック:
  - `src/theory.js` — 音名・コード・ダイアトニック・定番進行のデータと移調ロジック
  - `src/audio.js` — ピアノ風サウンドエンジン(単音・コード・進行の再生)
  - `src/quizzes.js` — クイズの問題ジェネレーター
  - `src/storage.js` — 進捗の localStorage 保存
